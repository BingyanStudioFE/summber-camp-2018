### 正确获取知识的姿势

> 你看百度的工程师都用在用 google，所以要什么百度自行车！

推荐时长：`0.8` 天

#### 任务目标

1. 成功翻越 GFW
2. 能够正常访问 Google、Facebook、Twitter、某hub（不要想歪！）等国外网站
3. （选做）成功申请并使用上 gmail
4. 修改默认搜索引擎为 google

#### 任务奖励

1. 经验值 `10 Exp`
2. 完成目标 3 额外获取 `5 Exp`
3. 瞬间提升的逼格

### 挑选一个适合自己的编辑器

> 工欲善其事，必先利其器

#### 任务目标

#### 任务奖励
